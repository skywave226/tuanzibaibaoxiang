# 更新说明

## 2025-07-17

### 站点

- 配置自定义域名：`https://tool.youxianmengguan.com/`
- 优化 SEO 静态页面生成，修复 SVG 图标截断问题
- 新增 51.la 访问统计

### 工具合并与整合

- **坐标转换**：合并坐标转换器与坐标变换器，统一为标签页界面
- **Cron 表达式**：合并多个 Cron 生成器，支持 5/6 段、随机示例、cURL 导入导出
- **闪卡**：合并闪卡工具，支持词性、例句、旧数据迁移
- **PDF 工具箱**：整合 PDF 合并、拆分、提取等功能

### 功能扩展

- **二维码生成器**：支持 Logo 叠加、批量生成、ZIP 打包、SVG 导出、生成历史
- **密码生成器**：支持口令短语模式、自定义符号集、排除易混淆字符、熵值评估
- **Base64 编解码**：支持 URL 安全 Base64、文件拖放转 Base64、图片/文本预览、原文件下载
- **URL 编解码**：支持 URL 参数解析、批量处理、自动判断编码方向
- **Hash 计算**：新增 MD5、文件哈希、HMAC、Base64/Hex 输出切换
- **时间戳转换**：支持时区选择、多种格式模板、批量转换
- **颜色转换**：新增 CMYK 输出、WCAG 对比度检查、配色方案、CSS 颜色名识别
- **单位换算**：新增速度、时间、数据、压强、能量等类别，增加市制/英制单位、交换、复制结果
- **UUID 生成器**：新增 UUID v1、多种输出格式（大写、无横线、URN、花括号）、NIL UUID、TXT 下载
- **大小写转换**：新增常量命名、路径命名、点分隔、URL Slug、反转大小写、交替大小写、字符统计
- **字数统计**：新增中文字符统计、阅读/朗读时间、词频排行、复制/粘贴/清空

### 其他增强

- **JSON 格式化**：新增 JSONPath 查询、Schema 校验
- **正则测试**：新增替换模式、正则解释
- **图片压缩**：新增批量处理、ZIP 下载
- **HTTP 请求测试**：新增环境变量、请求历史、cURL 导入导出
- **访问统计**：新增时间范围筛选、趋势图、CSV 导出
