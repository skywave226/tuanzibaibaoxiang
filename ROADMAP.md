# 团子百宝箱发展计划

> 本文档用于规划站点功能完善、新工具开发、页面补齐与设计优化方向。按优先级与可落地性排序。

---

## 一、功能完善计划

### 1.1 工具能力补齐（高优先级）

- [ ] **文本对比**：支持行内差异高亮、统一 diff 视图、导出 patch
- [ ] **代码格式化**：支持 HTML / CSS / JS / SQL / Markdown 等常见语言的格式化
- [ ] **图片工具链**：
  - 图片转 Base64 / Data URL
  - 图片裁剪 / 旋转 / 翻转
  - 图片水印（文字/图片水印）
  - 图片格式转换（PNG ↔ JPG ↔ WebP）
- [ ] **文件转换**：CSV ↔ JSON ↔ Markdown 表格 ↔ Excel
- [ ] **进制转换器**：增加位运算（与、或、非、异或、移位）和浮点二进制解析
- [ ] **JWT 解码器**：解析 Header / Payload / Signature，显示过期时间
- [ ] **正则测试**：增加常见正则预设库（邮箱、手机号、IP、URL 等）

### 1.2 交互与数据体验

- [ ] 工具状态持久化：保存用户最后一次输入/配置到 localStorage
- [ ] 全局收藏夹：用户可收藏常用工具到顶部快捷入口
- [ ] 最近使用：首页展示最近访问的 8 个工具
- [ ] 工具搜索增强：支持拼音、关键词同义词、分类过滤
- [ ] 批量操作通用化：抽象出可复用的批量输入/结果导出面板
- [ ] 结果区统一支持：复制、下载 TXT/JSON/CSV、清空

### 1.3 访问统计与分析

- [ ] 接入服务端统计（Umami / Plausible / Cloudflare Web Analytics）
- [ ] 统计工具增加来源分析、停留时长、跳出率
- [ ] 热门工具排行榜按周/月维度展示

## 二、新增工具计划

### 2.1 开发高频工具（第一批）

| 工具 | 核心功能 | 优先级 |
|------|---------|--------|
| 文本对比 | 两侧/合并 diff、高亮差异、导出 | 高 |
| Lorem Ipsum 生成器 | 生成指定段落/句子/单词数的占位文本 | 高 |
| HTML 实体编解码 | `&amp;` ↔ `&`，批量转换 | 高 |
| Markdown 编辑器 | 实时预览、导出 HTML、目录大纲 | 高 |
| JSON ↔ YAML ↔ TOML 转换 | 互转、错误提示 | 高 |
| 代码图片生成 | 代码转美观图片（Carbon 风格） | 中 |
| 占位图生成器 | 按尺寸/颜色/文字生成占位 SVG/PNG | 中 |
| Favicon 生成器 | 上传图片生成多尺寸 favicon | 中 |
| 网络测速 | 前端粗略测速、Ping 估算 | 中 |
| 浏览器信息检测 | UA 解析、屏幕/内核/语言/在线状态 | 中 |

### 2.2 生活与效率工具（第二批）

| 工具 | 核心功能 | 优先级 |
|------|---------|--------|

| 日期计算器 | 日期间隔、日期加减、工作日计算 | 高 |
| BMI 计算器 | 身体质量指数、标准范围参考 | 中 |
| 倒计时/番茄钟 | 全屏倒计时、音效提示 | 中 |

| 随机抽签/转盘 | 自定义选项、随机抽取 | 中 |
| 屏幕尺子 | 网页显示像素尺子 | 低 |

### 2.3 开发者工具（第三批）

| 工具 | 核心功能 | 优先级 |
|------|---------|--------|
| JSON 表格编辑器 | 类 Excel 编辑 JSON 数组 | 高 |

## 四、页面设计优化计划

### 4.1 视觉一致性

- [ ] 统一工具卡片的 padding、圆角、阴影、hover 动效
- [ ] 统一按钮层级：主操作使用 `type="primary"`，次操作使用默认/ghost
- [ ] 统一空状态、加载状态、错误状态图标和文案
- [ ] 统一表单输入区宽度与间距，避免不同工具布局错乱
- [ ] 为所有工具顶部增加一句用途说明，降低首次使用门槛

### 4.2 响应式与移动端

- [ ] 工具详情页在 768px 以下改为单列堆叠布局
- [ ] 首页网格在移动端改为 2 列或 1 列
- [ ] 侧边栏在移动端自动收起为抽屉菜单
- [ ] 大表格类工具（如 JSON 格式化结果）支持横向滚动

### 4.3 深色模式

- [ ] 修复部分工具在深色模式下的背景/边框/文字对比度问题
- [ ] 所有工具支持 `dark:` 类名覆盖
- [ ] 图表、图片预览、代码高亮区域适配深色模式

### 4.4 动效与微交互

- [ ] 页面切换增加淡入淡出过渡
- [ ] 工具卡片 hover 增加轻微上浮和阴影加深
- [ ] 复制成功增加全局 Toast 反馈
- [ ] 结果区域加载/生成时增加骨架屏或 spinner

### 4.5 性能优化

- [ ] 工具路由按需懒加载，降低首屏 JS 体积
- [ ] 大型库（xlsx、ajv、pdf-lib、jsQR 等）全部改为动态导入
- [ ] SEO 静态页面构建提速，考虑增量生成
- [ ] 图片资源懒加载、SVG 图标统一内联或雪碧图

### 4.6 品牌与文案

- [ ] 统一工具命名风格（动词/名词一致）
- [ ] 优化工具描述，突出核心卖点，便于搜索引擎展示
- [ ] 为每个工具补充 5-10 个关键词标签
- [ ] 首页增加品牌 slogan 和信任背书（工具数量、开源等）

---

## 五、近期建议执行顺序

1. **立刻做**：修复移动端响应式、统一空状态/复制反馈
2. **本周内**：补齐文本对比、HTML 实体编解码、Lorem Ipsum、Markdown 编辑器
3. **本月内**：新增房贷计算器、日期计算器、JSON ↔ YAML 转换、JWT 解码器
4. **持续做**：根据访问统计和首页点击热区，迭代高流量工具的细节体验
